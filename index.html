Connect Your Google Sheet

Paste your Apps Script Web App URL below. This is how the dashboard reads your data live from Google Sheets.

Setup Guide
1

Create a new Google Sheet

Go to sheets.google.com and create a blank spreadsheet. Name it something like Flip Tracker.

2

Open Apps Script

In your Sheet, go to Extensions → Apps Script. Delete any existing code and paste in the entire contents of APPS_SCRIPT.gs (included in your download).

3

Run the setup function

In Apps Script, select setupSheets from the function dropdown and click ▶ Run. This creates all your tabs (Accounts, Expenses, Inventory, Dashboard, GP Tracker) with the correct headers. Grant permissions when asked.

4

Deploy as Web App

Click Deploy → New deployment. Choose type: Web app. Set Execute as: Me and Who has access: Anyone. Click Deploy and copy the Web App URL.

5

Paste the URL above

Paste your Web App URL into the field at the top of this page and click Save & Test. If it says "Connected!" you're done. Start adding accounts to your Accounts sheet!

6

Host this dashboard (optional but recommended)

Upload index.html to a free GitHub Pages repo so you can access it from anywhere. Your URL will look like: yourusername.github.io/flip-tracker

7

Sheet column reference

Accounts: Name | Type | Description | Asking Price | Status (Building/Listed/Sold) | Sold Price | Date Sold

Expenses: Account Name | Category | Description | Amount | Date | Notes

Types: Pure / HCIM / Ironman / Main / Other
